# TeslaBot
TeslaBot is Penetration Tool That Exploits Some Simple Web vulns and Some Other Features

## Requirements
* googlesearch
* requests
* colorama
* Python3

## Installation



```bash
apt-get install python3
pip3 install googlesearch
pip3 install requests
pip3 install  colorama
```
<h3>OR:</h3>
```bash
chmod +x install.sh
./install.sh
```
